1. Please edit your Makefile for your OS.
Default settings are for 64 bit mac x os.
If you are using 32 bit linux, please comment the default settings
and  uncomment the options for your OS.
2. Type: make
to build the project and generate .out executable file named "prep"
3. Type: make clean
to remove the .o ducoments
4. Type: ./prep.out
to run the executable file prep
